### 关于树洞
「鲸鱼的树洞」之域名是 [www.jinng.cn](https://www.jinng.cn)，jinng 是鲸鱼的吴语拼音（~~因为普通话拼音被注册了~~），可以去「[吳語學堂](https://www.wugniu.com/search?char=%E9%B2%B8%E9%B1%BC&table=suzhou_zi)」听听发音。

「鲸鱼的树洞」由 [Hugo](https://gohugo.io) 生成，存储在 [GitHub](https://github.com) 上，由以下三个部分组成：
- 「[树干](https://github.com/tseujinng/shugan)」：网站的 Hugo 框架，同时负责运行 GitHub Actions。
- 「[树洞](https://github.com/tseujinng/shudong)」：网站的文字内容，如果你发现错别字，可以在这里提交。
- 「[树叶](https://github.com/tseujinng/shuye)」：网站的 Hugo 主题，基于 [simple.css](https://simplecss.org/)。

「鲸鱼的树洞」通过 GitHub Actions 生成后会上传至腾讯云 COS，形成你现在正在访问的源站。
